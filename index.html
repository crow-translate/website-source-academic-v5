Crow Translate

A simple and lightweight translator that allows you to translate and speak text using Google, Yandex, Bing, LibreTranslate and Lingva.

Star

Install Source code

Latest release

Features

Speed

The program is written in C++ / Qt and consumes ~20 MB of RAM.

Open source

Crow Translate is licensed under the GPL v3 license, which means you can use and modify it for free.

Many languages

Thanks to Google, Yandex and Bing, you can translate into 117 different languages.

CLI interface

You can translate text directly in the terminal.

Selection / OCR

Translate and speak text from screen or selection

Cross-platform

Available for Linux and Windows.
